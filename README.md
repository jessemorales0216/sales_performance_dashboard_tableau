# Retail Sales Dashboard (Tableau)

## Overview

This project analyzes retail and warehouse sales data from 2017 to 2020 using Tableau. The dashboard highlights sales trends, top-performing products, and category performance.

## Tools Used

* Tableau Public
* Excel / CSV data

## Key Insights

* Beer category drives the majority of total sales (~$7M)
* Sales show steady growth after 2019
* Top products are dominated by Corona and Heineken

## Features

* KPI metrics (Total Sales, Avg Monthly Sales)
* Sales trend over time
* Top 10 products visualization
* Category performance breakdown
* Interactive filter by item type
